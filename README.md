# Pokedex

Projeto de Pokedex feito com Angular/Typescript para treinamento de consumo de api.
Treinadores pokemon e programadores frontend podem acompanhar a versão live do projeto em [https://regismrs.github.io/Pokedex/](https://regismrs.github.io/Pokedex/).

## Características

1. Consumo de open api com dados referentes a Pokemon
2. Paginação para listagem dos +1200 pokemons
3. Renderização seletiva enquanto o recurso não estiver dísponivel
